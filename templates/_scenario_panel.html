File structure

import
Load another Lean file — here, the shared medical vocabulary.
namespace … end
Open a named section; every declaration inside gets that prefix.
open
Refer to names in a namespace without repeating the prefix.

Declarations

axiom
A starting fact Lean accepts without proof — a guideline or a chart finding.
theorem
A claim Lean accepts only after mechanically checking its proof.
X : T
Read : as “is a”. JohnDoe : Patient = “JohnDoe is a Patient”.
X := Y
Read := as “is defined as”. Names X after the content on the right.

Proof tactics

by
Switch into step-by-step tactic mode — a recipe for building the proof.
unfold
“Replace this name with its definition” so its structure is visible.
apply
“Use this lemma; I'll prove its premises next” as fresh sub-goals.
exact
“Here is a term that is exactly the proof required — close the step.”
·
Bullet that starts the next sub-goal left open by an earlier tactic.

Contradiction

And.intro
Constructor for “P and Q” — supply a proof of P and a proof of Q. Used here to assemble the Indicated ∧ Contraindicated collision.
False
The proposition that is never true. Proving it means the axioms contradict each other.
absurd
The final theorem in each scenario — its claim is False, so its proof is the contradiction.

Meta-command: lists every axiom the proof actually relied on, so the host can rule out sorry placeholders.
